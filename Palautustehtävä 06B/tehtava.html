Tehtävää

Create a new html document and name it third.html

Make the layout to be the same as in second.html and index.html (coding excercises 03B, 04B and 05B)

Create some content to third.html.

Modify third.html-page and create a new vertical navigation to that site and place on the left side of your page. Use the float-property to do this.

Place the section div of your website, next to your new vertical navigation, so you are able to set the content to be on two different columns. You can define width of this columns yourself by using the width-property.

Add two new links to your vertical navigation. Make the links to go to some external site (for example google.com etc...) Define these links to open in New Tab.

Add new link to your main navigation which leads to the third.html-page. (Remember to modify all the html documents, to make your navigation work properly)

Define margin and padding-properties to different elements and try to enhance design of your site. For example: Add some padding to text-elements and your text will look better when it is not hugging the side of a div.

And lastly, lets modify the main navigation and create a pseudo-class effect to your main navigation links. The effect you add, should change the color of the link to red when you are hovering over it.
